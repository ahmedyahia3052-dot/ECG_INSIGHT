Build a production-ready SaaS platform called "ECG Insight".

Project Goal:
ECG Insight is an AI-powered ECG interpretation and clinical decision support platform designed for doctors, cardiology residents, medical students, and healthcare organizations.

Tech Stack:
- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS + Shadcn UI
- State Management: Zustand
- Data Fetching: TanStack React Query
- Forms: React Hook Form + Zod
- Routing: React Router DOM
- Icons: Lucide React
- Charts: Recharts
- Backend-ready architecture
- Fully responsive design
- Dark and Light Mode

Design Requirements:
- Premium modern medical SaaS UI
- Professional healthcare color palette (teal, cyan, white, dark gray)
- Enterprise-grade design
- Smooth animations and transitions
- Mobile-first responsive layout
